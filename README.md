## Sobre o projeto
Este projeto é um aplicativo de finanças desenvolvido em Java com foco no controle básico de entradas e saídas de dinheiro pelo terminal.
A proposta principal é permitir que o usuário registre receitas e despesas, visualize as transações cadastradas, consulte o saldo atual e remova movimentações quando necessário.

## Funcionalidades
Atualmente, o sistema permite:

- Adicionar uma **receita**
- Adicionar uma **despesa**
- Listar transações cadastradas
- Verificar o saldo atual
- Remover uma transação
- Encerrar o programa

## Estrutura do projeto
O projeto foi construído em Java e executado via terminal, com classes separadas para representar a lógica do sistema.

Exemplo das classes principais:

- `Main` → controla o menu e a interação com o usuário
- `Carteira` → gerencia a lista de transações
- `Transacao` → classe abstrata base
- `Receita` → representa entradas de dinheiro
- `Despesa` → representa saídas de dinheiro
